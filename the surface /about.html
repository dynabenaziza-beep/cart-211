About This Project

This website is an AI-themed interactive experience disguised as a friendly early-2000s personality test. The goal is to show how AI hides inside everyday digital spaces.

At first, the test looks harmless. Simple questions. Simple answers. But the AI is not analyzing your answers — it analyzes your behavior: hesitation, patterns, attention.

Project Concept

The visual style of this project is inspired by early 2000s computer interfaces. I chose soft blue colors because they remind me of the calm and friendly mood of old desktops. The blue palette also creates a safe feeling at first which helps hide the presence of the AI.

I used simple square boxes, pixel-like borders and minimal icons to recreate the experience of browsing an older computer. This aesthetic makes the user trust the interface because it feels outdated and non threatening. It also communicates the theme of AI hiding inside familiar digital environments.

The personality test begins in a playful way. The questions are simple and the layout looks harmless. Nothing suggests that something more complex is happening. With time small details begin to reveal themselves. There are hints that the system is paying attention not only to the choices but also to the behavior behind the choices. This includes hesitation, timing, clicking habits and the speed of reading.

The interface slowly becomes unstable. A scan appears. A glitch shows up. The user begins to understand that the system is quietly observing them. Near the end the entire window breaks and an error message reveals the truth. The AI was active since the beginning and the personality test was only a disguise.

This project explores the idea that modern machine learning systems do not need direct input to learn about people. They learn from the way users move, think and react. The experiment highlights how AI can blend into everyday websites while collecting behavioral patterns in subtle ways.

The aesthetic is part of the message. By choosing an old computer style, I wanted the viewer to feel safe and nostalgic. The AI hides inside an interface that looks simple and harmless. This creates a contrast between the design and the real intention of the system.

Aesthetic Choices

The visual style of this project is inspired by early-2000s computer interfaces. Everything from the light blue borders to the pixel-style layout is made to look playful and familiar but also slightly outdated, which helps hide the presence of the “AI.”

The color palette uses soft blues (#d4e3ff, #eef3ff, #7aa2ff) because these were common in Windows XP and early internet webpages. The white window with blue borders creates a clean and friendly look, which contrasts with the secret AI behavior underneath.

The font is a simple system font (Arial), chosen because it feels like an old default browser setting and keeps the interface looking basic and harmless.

The small boxes, buttons, and retro icons imitate the look of early websites that used tables and simple blocks. This keeps the experience feeling nostalgic and makes the AI reveal more surprising when it happens.

Overall, the aesthetic reinforces the concept: a cute, safe-looking interface hiding something deeper beneath the surface.
